# Regular Expressions

This project is part of the ALU Software Engineering curriculum.

## Description

The goal of this project is to learn and practice regular expressions (regex) using the Oniguruma regular expression library, which is the default regex engine used by Ruby.

Each task consists of writing a Ruby script that uses a specific regular expression to match a given pattern.

## Requirements

* Ubuntu 20.04 LTS
* Ruby
* All scripts are executable.
* The first line of every script is:

```ruby
#!/usr/bin/env ruby
```
